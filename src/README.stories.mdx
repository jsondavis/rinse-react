<h1>Welcome to {pkg.name} Component Library</h1>

Add new components directory or combinations of components and when you add a story for them they should render in the sidebar here.
Out of the box this setup assumes you'll be using styled-components and jest. And that you're ok with eslint and prettier being run on commit.

Ideally you'd begin with the smallest possible components then combine those into a feature.  Finally combine features to create a page. At some level wrap the component(s) in a ThemeProvider so that they can reference theme values.

See ExampleButton, ExampleFeature and ExamplePage for one way of accomplishing this.
